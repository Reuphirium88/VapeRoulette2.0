# Vape Roulette 2.0 - Mini App Frontend

This repository contains the frontend for the Vape Roulette 2.0 Mini App.

## Status

This frontend is currently under development.

**Deployment:**
The frontend is intended to be published on GitHub Pages.

**Data:**
Currently, the application operates with **mock data** for user XP and lootbox contents. There is no real API integration yet.

## Getting Started (Local Development)

1.  Clone this repository.
2.  Open `index.html` in your web browser.

## Features (Current)

*   Display of user XP balance (mock data).
*   List of lootboxes with their XP costs and potential prizes (mock data).
*   "Open" buttons for each lootbox (functional with mock data, deducting XP and showing a random prize).
*   Basic integration with Telegram WebApp (expands and sets background color if running within Telegram).
